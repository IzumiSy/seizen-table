## Quick Start

Here's a basic DataTable in action:

<DemoPreview demo="basic" height={300} client:load />

```tsx
import { useDataTable, DataTable } from "@izumisy/seizen-datatable-react";

const columns = [
  { accessorKey: "name", header: "Name" },
  { accessorKey: "age", header: "Age" },
  { accessorKey: "email", header: "Email" },
];

const data = [
  { name: "John Doe", age: 30, email: "john@example.com" },
  { name: "Jane Smith", age: 25, email: "jane@example.com" },
  { name: "Bob Wilson", age: 35, email: "bob@example.com" },
];

function BasicDemo() {
  const table = useDataTable({ data, columns });

  return <DataTable table={table} />;
}
```

## Enable Features

Seizen DataTable follows a **plugin-first architecture** â€” all features, including official ones, are provided as plugins. This keeps the core lightweight while allowing you to add only what you need.

```tsx
import { useDataTable, DataTable } from "@izumisy/seizen-datatable-react";
import { RowDetail } from "@izumisy/seizen-datatable-plugins/row-detail";
import { Filter } from "@izumisy/seizen-datatable-plugins/filter";

function MyTable() {
  const table = useDataTable({
    data,
    columns,
    plugins: [
      RowDetail.configure({ ... }),
      Filter.configure({ ... }),
    ],
  });

  return <DataTable table={table} />;
}
```

### Official Plugins

| Plugin | Description |
|--------|-------------|
| [Row Detail](/seizen-table/features/row-detail/) | Expandable panel to show detailed row information |
| [Filtering](/seizen-table/features/filtering/) | Build complex filters with an intuitive UI |
| [Column Control](/seizen-table/features/column-control/) | Toggle column visibility and reorder columns |
| [Data Export](/seizen-table/features/data-export/) | Export table data to CSV, JSON, and more |

Learn more about the [Plugin System](/seizen-table/guides/1-plugins/) to create your own custom plugins.
